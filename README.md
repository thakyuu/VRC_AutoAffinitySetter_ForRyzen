## なにこれ

Ryzenの場合、なんかVRChatで利用するCPUを単一CCX内のみにするとパフォーマンスがあがるらしい

毎回設定するのめんどくさいから自動で設定するツール

## どうやって使うの

https://github.com/thakyuu/VRC_AutoAffinitySetter_ForRyzen/releases/tag/0.0.1

VRChat起動後に実行するだけ

AppBinderとかでVRChat起動時に自動実行させると何も考えずに使えて便利です
